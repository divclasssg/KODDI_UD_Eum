# 의료 취약군 문진 보조 앱 설계서

작성일: 2026-07-10

## 1. 만들려는 것

병원에서 증상을 말하기 어려운 사람이 있다.

예를 들면:

- 말로 길게 설명하기 어려운 사람
- 글을 읽거나 쓰기 어려운 사람
- 병원에서 긴장해서 증상을 잘 말하지 못하는 사람
- 고령자
- 발달장애인
- 실어증 등으로 말하기가 어려운 사람

이 앱은 이런 사람이 병원에 가기 전에 증상을 쉽게 기록하도록 돕는다.

사용자는 글, 음성, 사진, 선택 버튼으로 증상을 입력한다.  
AI는 필요한 내용을 짧게 물어본다.  
마지막에는 의사가 보기 쉬운 요약을 만든다.

이 앱은 병을 진단하지 않는다.  
이 앱은 치료 방법을 알려주지 않는다.  
이 앱은 의사를 대신하지 않는다.

AI가 만든 요약은 의사가 참고하는 초안이다.

## 3. 꼭 필요한 화면

UI는 사용자가 Figma를 보고 직접 만든다.  
개발자는 그 UI에 저장 기능, AI 기능, 버튼 동작을 연결한다.

### 반드시 필요한 화면

| 순서 | 화면 | 하는 일 | Figma 기준 |
|---|---|---|---|
| 1 | 온보딩/동의 화면 | AI 사용 안내, 의료진 참고용 안내, 회원가입 없음 안내 | `온보딩_1`, `온보딩_2`, `온보딩_3`, `온보딩_4` |
| 2 | 기본 프로필 입력 | 나이 또는 생년월일, 성별 입력 | `온보딩_4`, `2_2_개인정보 수정` |
| 3 | 의료정보 입력/수정 | 키, 몸무게, 복용약, 알레르기, 가족력, 기존 질환, 흡연/음주 입력 | `2_1_기본 프로필_데이터X`, `2_3_의료 정보 수정_키/몸무게`, `3_1_의료정보추가`, `2_4`, `2_5`, `2_6` |
| 4 | 홈 화면 | 문진 시작, 최근 기록 보기 | `Home_Main` |
| 5 | AI 문진 화면 | 글, 음성, 사진, 영상, 선택 버튼으로 답하기 | `Chat_01_Entry`, `Chat_02`, `Chat_04`, `Chat_05`, `Chat_08`, `Chat_09`, `Chat_10` |
| 6 | 저장 확인 화면 | 새 의료정보를 저장할지 물어보기 | Figma에 없으면 아래에서 올라오는 창 또는 팝업 필요 |
| 7 | 의료진용 요약 화면 | 의사에게 보여줄 S/O 요약 보기 | `기록 - 자세히보기`, `iPhone 16 - 65` |
| 8 | 기록 목록 화면 | 이전 문진 기록 다시 열기 | `기록` |
| 9 | 설정 화면 | 잠금 설정, 기록 삭제, 의료정보 삭제 | Figma에 없으면 새로 필요 |

### 꼭 필요한 상태 표시

아래 상황은 화면 안에서 반드시 보여줘야 한다.

- AI가 답을 만드는 중
- AI 연결 실패
- Hugging Face 무료 사용량 초과
- 다시 시도 버튼
- AI가 안 될 때 수동으로 요약 보기
- 음성 인식 중
- 음성 인식 실패
- 사진 첨부 완료
- 사진 첨부 실패
- 영상 첨부 완료
- `예`
- `아니오`
- `모르겠음`
- 새 의료정보 저장 확인
- 저장 완료
- 저장 실패

### Next.js 화면 주소

Next.js에서는 아래 주소를 기본으로 만든다.

```text
/
/onboarding
/home
/interview/new
/records
/records/[id]
/profile
/profile/medical
/settings
```

`기록 - 검색하기` 화면은 처음에는 꼭 만들지 않아도 된다.  
가장 중요한 흐름은 이것이다:

문진 작성 → 필요한 의료정보 질문 → 저장 여부 확인 → S/O 요약 생성 → 병원에서 다시 열기

## 4. 이번에 만드는 범위

### 만든다

- 모바일 앱처럼 보이는 웹앱
- Next.js로 구현
- 회원가입 없이 시작
- 선택적 잠금 설정
- 기본 프로필과 의료정보를 사용자 기기에 저장
- 글 입력
- 음성 입력
- 사진 첨부
- 사진을 AI에 전달
- 영상 파일 첨부
- 선택 버튼으로 증상 입력
- AI 문진
- 기본 4~5개 중심 질문
- Hugging Face로 MedGemma 실제 연결
- 의료진용 S/O 요약 생성
- 응급 신호가 보이면 안전 안내
- 접근성을 고려한 화면

### 만들지 않는다

- 회원가입
- 로그인
- 별도 데이터베이스
- 병원 EMR 연결
- 진단명 추론
- 치료 방법 추천
- 처방 추천
- 영상 분석
- 웨어러블 자동 연결
- 유료 GPU
- Hugging Face PRO
- Vertex AI
- 유료 Inference Endpoint
