## 결정 사항

- 문서가 충돌하면 사용자 지시 → 2026-07-16 계획 → 업데이트된 Google Docs → 과거 설계 순으로 판단한다.
- 상위 문서는 현황과 탐색에 집중하고 상세 내용은 상대 링크로 참조한다.
- Markdown 파일은 최대 200줄, 권장 분리 시점은 약 180줄로 관리한다.
- mock 연동과 실제 MedGemma 연동의 완료 상태를 분리해 기록한다.
- 애플리케이션 스타일은 Sass/SCSS만 직접 사용한다. 브라우저 접두사는 Next.js 기본 `postcss-preset-env` Autoprefixer가 처리하며, `postcss`·`autoprefixer` 직접 의존성이나 `postcss.config.*`는 추가하지 않는다. Next.js의 전이 `postcss`는 프레임워크 내부 의존성으로만 유지한다.
