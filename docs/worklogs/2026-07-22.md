## Modal actual·복구

- 실제 환자 정보·음성·마이크·STT 없이 합성 Persona `kim`의 선택 답변 한 번만 사용했다.
- 첫 실행은 actual 서버의 Route 전용 Origin·HMAC 누락 때문에 GPU 호출 전 HTTP 503으로 실패했다.
- test server 환경을 보정한 mock Route가 HTTP 200임을 확인한 뒤 실제 질문 1회만 다시 실행했다.
- 실제 요청은 Node guard와 Modal proxy 인증을 통과했고 T4가 cold load 후 17토큰을 생성했다.
- 생성 뒤 웹 응답이 Next provider 75초 안에 반환되지 않아 `/api/ai/question`은 HTTP 502 fallback으로 종료됐다.
- 비용 우선 원칙에 따라 추가 GPU 질문·요약·전체 actual suite는 실행하지 않았다.
- `MEDGEMMA_ACTUAL_DISABLED=1` Secret 재설정과 재배포를 완료했다.
- 인증된 status-only 합성 probe에서 HTTP 503, 이후 `main` 실행 container 0을 확인했다.
- 운영 provider 기본값과 direct cold 성능 기준 75초는 유지하고, Next·네트워크 왕복을 포함하는 browser actual server만 기존 상한 85초를 사용하도록 config 계약을 RED→GREEN으로 보정했다.
- 사용자 승인 뒤 합성 `kim` Persona 질문 1회만 재검증했고 `/api/ai/question` HTTP 200과 새 질문 표시가 58.7초에 통과했다.
- 성공 직후 kill switch `1`을 재배포하고 인증된 status-only 합성 요청의 HTTP 503과 실행 container 0을 확인했다.
